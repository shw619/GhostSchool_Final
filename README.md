# Project
1. 프로젝트명

   괴담학교 : 어두운 복도

2. 게임 개요

   장르: 공포, 탈출

   개발 엔진: Unity 2D

   플랫폼: PC

   시점: 2D 횡스크롤 3인칭 사이드뷰

   타겟층: 공포 게임을 좋아하는 10~20대

   특징: 시각이 제한된 플레이어가 공포 요소인 소리를 통해 귀신을 피하고 제한 시간 안에 교재를 모아 학교를 탈출하는 게임
